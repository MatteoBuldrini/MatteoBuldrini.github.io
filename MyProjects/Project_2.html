A/B Testing

In this project I performed an A/B test to understand whether the implementation of the new web page for an e-commerce website generates a significant increase in the conversion rate of users compared to the old page.

The notebook is divided in three main sections:

  1. Probability: data cleaning and calculation of the probability of conversion;
  2. A/B Test: definition of the hypotheses and actual test, using first the sampling distribution and then Z-test approach;
  3. Model: creation of a logistic regression model.
