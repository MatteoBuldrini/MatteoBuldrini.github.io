My Certifications

Be curious and never stop learning is my motto. Here is a list of certifications I completed which I highly recommend. If you are interested, click to view the full certificate.

Data Analyst Nanodegree

An online program that covers Inferential statistics, Probability and the entire Data Analysis process (from data collection to data visualization) in depth, using Python.

SQL Nanodegree Program

An online program that covers how to write and optimize advanced queries, as well as how to design efficient database schemas for relational databases.

Microsoft Power BI Data Analyst

An official Microsoft certification that validates your ability to prepare, model visualize and analyze the data using Power BI.

IBM Data Science Professional Certificate

An online Professional Certificate offered by IBM which covers a wide array of data science topics including SQL, Python, data visualization, data analysis and machine learning.

Data Engineer in Python

An online program which covers data engineering topics, such as cloud computing, Python programming, version control with Git and introduces data pipelines and Airflow for complex data workflows.

Python for Finance: Investment Fundamentals & Data Analytics

An online course which covers finance topics such as risk and return of investment portfolios, CAPM and regression analysis and their practical application in projects using Python.

Data Visualization with Tableau

An online Specialization which covers the creation of reports and dashboards with Tableau, one of the leading data visualization software.

The Web Developer Bootcamp

An online Bootcamp which covers many tools and technologies including HTML, CSS, JavaScript, NodeJS, ExpressJS, MongoDB.
